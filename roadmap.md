# Integrated Growth & Monetisation Roadmap — FreeTraderHub.com

> **Part of an integrated strategy. Read alongside:**

> - `revenue-model.md` — projections and trust rules

**How this roadmap works:**
Traffic milestones are listed alongside revenue milestones in each phase.
They are not separate tracks — traffic gates revenue. If a traffic milestone
is missed, the revenue milestone that depends on it shifts accordingly.

## Phase 1 — Foundation (Days 0–30)
**Goal:** First dollar of revenue + traffic engine started + demand validated.

### Revenue Track

**Week 1 — Affiliate Setup (2–3 hours)**
- [ ] Apply to FTMO affiliate program
- [ ] Apply to FundedNext affiliate program
- [ ] Apply to The5ers affiliate program (priority: 20–40% + lifetime share)
- [ ] Build one "Firm Comparison Table" with labelled affiliate links
- [ ] Audit existing pages for natural affiliate link placement

**Week 2 — Demand Validation (no code)**
- [ ] Add ghost "Save session" button to Position Sizer (modal → Pro waitlist email capture)
- [ ] Publish Tally survey: "Would you pay $7/month for rule-change alerts?"
- [ ] Add course waitlist CTA to most-trafficked page
- [ ] Success signal: >50 combined waitlist signups by Day 30

### Traffic Track

**Week 1 — SEO Foundation**
- [ ] Submit XML sitemap to Google Search Console
- [ ] Identify top 20 programmatic SEO target pages from rule library
  (see `traffic-growth-plan.md` Section 1A for full target list)
- [ ] Add FAQ schema to top 5 rule pages
- [ ] Measure baseline: record current monthly unique visitors

**Week 2 — Community Activation**
- [ ] Join 10 relevant Discord servers (FTMO-focused, Forex, prop firm general)
  (see `traffic-growth-plan.md` Section 2 for full server target list)
- [ ] Participate genuinely for 1 week before posting any tool links
- [ ] Join r/PropFirms, r/Forex — answer 3 rule questions using site content

**Week 3–4 — First Content Push**
- [ ] Publish 2 long-form editorial SEO pieces:
  1. "Complete guide to not failing your FTMO challenge" (course waitlist CTA at end)
  2. "FTMO vs FundedNext 2026 — independent comparison" (affiliate links, labelled)
- [ ] Build 5 programmatic rule pages (highest-search-volume rules first)
- [ ] Add email capture CTAs to tool pages: "Get rule change alerts — enter email"
- [ ] Set up email platform (Beehiiv or ConvertKit — free tier)

**Phase 1 Traffic Target:** Baseline measured + SEO content indexed + 50+ email list subscribers
**Phase 1 Revenue Target:** $300–$800 (first affiliate commissions by Day 15)
**Phase 1 Budget:** < $50

## Phase 2 — Build What's Validated (Days 30–90)
**Goal:** Two recurring revenue streams live. Email list at 500+. Organic traffic doubled.

**Traffic gates revenue in this phase:**
- Stream 3 (alerts) launches when email list hits 250
- Stream 2 (Pro tools) launches when waitlist hits 50 signups
- Stream 4 (course) pre-sell waits for email list at 1,000

### Revenue Track

**Milestone 2A — Rule Alert Tier (Days 30–50)**
*Prerequisite: >80 survey positive responses AND email list at 100+*

- [ ] Set up Beehiiv paid subscription tier at $7/month
  (OR: simple webhook monitoring rule library git diffs → email dispatch)
- [ ] Write alert email template: diff format + plain-English explanation + tool link
- [ ] Email full list: "Rule Change Alerts are live — first month free for early subscribers"
- [ ] Add upgrade CTA to all rule library pages: "Get alerted when this rule changes"

Expected: 50–100 paying subscribers at $7/month = **$350–$700/month MRR**

**Milestone 2B — Pro Tool Tier (Days 45–75)**
*Prerequisite: >50 waitlist signups*

- [ ] Implement Clerk auth (free to 10K MAU)
- [ ] Integrate Stripe billing ($12/month, $99/year)
- [ ] Gate "Save session" behind Pro
- [ ] Build multi-firm comparison view (highest-demand Pro feature)
- [ ] Launch to waitlist: "Pro is live — first 50 get 30% off forever"

Expected: 60–100 subscribers at $12/month = **$720–$1,200/month MRR**

### Traffic Track

**Milestone 2C — SEO Acceleration (Days 30–90, ongoing)**
- [ ] Publish 8–10 programmatic rule pages (one per major rule type per firm)
- [ ] Build "FTMO vs FundedNext" full comparison page (highest-volume keyword in niche)
- [ ] Build "Best prop firm for swing trading" guide (affiliate-heavy, SEO optimised)
- [ ] Submit updated sitemap; verify Google Search Console coverage
- [ ] Target: **5,000+ monthly organic visitors by Day 90**

**Milestone 2D — Community Growth**
- [ ] Share tools contextually in 5+ Discord servers (not spam — answering real questions)
- [ ] Post 2 Reddit threads with data/tool showcase format
  (see `traffic-growth-plan.md` Section 3 for format examples)
- [ ] Build trackable invite links for 3 Discord servers (UTM or referral codes)
- [ ] Target: **1,000+ Discord/Reddit referral sessions by Day 90**

**Milestone 2E — Email List Growth**
- [ ] Publish "2026 Prop Firm Rule Cheat Sheet" as gated PDF (email capture landing page)
- [ ] Share cheat sheet in Discord and Reddit alongside tool mentions
- [ ] Add exit-intent popup to tools pages: "Get alerted when FTMO changes rules"
- [ ] Start 5-email nurture sequence for new subscribers
  (see `traffic-growth-plan.md` Channel 6 for sequence detail)
- [ ] Target: **500 email subscribers by Day 90**

**Phase 2 Traffic Targets:**

| KPI | Day 60 Target | Day 90 Target |
|---|---|---|
| Monthly organic visitors | +30% from baseline | +100% from baseline |
| Discord/Reddit referral sessions | 300 | 1,000 |
| Email list size | 150 | 500 |
| Email open rate | 45%+ | 40%+ |

**Phase 2 Revenue Target:** $2,000–$3,000/month MRR by Day 90
**Phase 2 Budget:** $500–$1,500 (auth/billing tooling + possible freelance dev 10–15h)

## Phase 3 — Scale Winners, Cut Losers (Days 90–180)
**Goal:** $5,000+/month. 10,000+ monthly visitors. 2,500+ email subscribers.

**Traffic gates revenue in this phase:**
- Course pre-sell requires email list at 1,000+ (move to Day 90 only if already there)
- Sponsored listings require 8,000+ monthly visitors (likely Month 4–5)
- Educator partnerships scale affiliate and course revenue simultaneously

### Revenue Track

**Milestone 3A — Challenge Prep Course (Days 90–120)**
*Prerequisite: course waitlist >30 AND email list >1,000*

- [ ] Pre-sell to waitlist at $97 introductory price (minimum 20 pre-sales before recording)
- [ ] Record 5 modules (Loom video + written transcript ≈ 8–10 hours total)
- [ ] Host on Gumroad or Podia (no custom platform)
- [ ] 7-day launch email sequence to full list
- [ ] Offer educators 30–40% affiliate commission for course referrals

Expected: $2,000–$4,000 launch + **$500–$1,500/month ongoing**

**Milestone 3B — Sponsored Listings (Days 120–180)**
*Prerequisite: 8,000+ monthly visitors AND 1,000+ email subscribers*

- [ ] Build media kit: traffic, list size, audience demographics, engagement rates
- [ ] Cold email 10 smaller/newer prop firms with free 30-day trial offer
- [ ] Cap at 3 featured firms; explicit contract that editorial data is independent
- [ ] Annual prepay option: $2,500/year (30% discount)

Expected: **$600–$900/month** from 2–3 sponsors

### Traffic Track

**Milestone 3C — Expand Firm Coverage (Days 90–150)**
The rule library is the core content moat and SEO engine. Expanding it multiplies
SEO surface area, affiliate partners, and alert subscriber value simultaneously.

- [ ] Add MyFundedFX, Apex Trader Funding, Funded Engineer to rule library
- [ ] Apply to affiliate programs for each new firm
- [ ] Build dedicated comparison pages for each new firm pairing
- [ ] Expected: **+30–50% organic traffic; +20–40% affiliate revenue**

**Milestone 3D — YouTube / Short-Form Video (Days 90–150)**
*(see `traffic-growth-plan.md` Channel 4 for full playbook)*

- [ ] Identify 5–10 trading YouTubers covering prop firms (10K–200K subscribers)
- [ ] Outreach: offer free Pro account + 30% affiliate share on referrals
- [ ] OR: Record 2 tool walkthrough videos for owned channel (screen record + voiceover)
- [ ] Expected: **1,000–5,000 monthly sessions from video by Month 5**

**Milestone 3E — Educator Partnerships (Days 90–120)**
*(see `traffic-growth-plan.md` Channel 5 for full playbook)*

- [ ] Identify 10 trading educators covering prop firm challenges
- [ ] Outreach: free Pro account + co-branded content offer
- [ ] Build co-branded landing page for each partner with trackable link
- [ ] Expected: **500–3,000 qualified monthly sessions per active partnership**

**Phase 3 Traffic Targets:**

| KPI | Month 4 | Month 6 |
|---|---|---|
| Monthly unique visitors | 8,000+ | 15,000+ |
| Monthly organic visitors | 6,000+ | 12,000+ |
| Email list size | 1,500 | 2,500+ |
| YouTube/video referral sessions | 300 | 1,500 |
| Educator referral sessions | 500 | 2,000+ |

**Phase 3 Revenue Target:** $5,000–$7,000/month by Day 180
**Phase 3 Budget:** $500–$1,000 (course tooling + freelance content)

## Milestone 3F — Cut What's Not Working (Day 90 Review)

| Metric | Cut Signal | Scale Signal | Action |
|---|---|---|---|
| Affiliate CTR | < 1% on comparison pages | > 3% | Cut: rewrite page or remove links |
| Pro trial-to-paid conversion | < 20% | > 35% | Cut: reprice or rebuild feature gate |
| Alert churn (monthly) | > 10% | < 5% | Cut: improve alert quality/frequency |
| Course completion rate | < 30% | > 60% | Cut: restructure course delivery |
| Discord referral sessions | < 200/month | > 1,000/month | Cut: focus on higher-performing servers |

Kill underperforming streams. Double down on what's converting. Do not run
five streams at half-effort — run two streams at full effort.

## Risk Register

| Risk | Likelihood | Impact | Mitigation |
|---|---|---|---|
| Prop firm changes affiliate terms | Medium | Medium | Diversify to 5+ firms by Month 3 |
| Trust damage from perceived bias | Low | High | Disclose everything; rule data independent |
| Pro tool low conversion | Medium | Medium | Validate with waitlist before building |
| SEO traffic slow to materialise | Medium | Medium | Discord/email list provides revenue bridge |
| Prop firm industry contraction | Low | High | Expand to futures/crypto prop firms |
| Course content goes stale | Medium | Low | Frame around principles, not firm-specific rules |
| Founder time constraint | High | Medium | Phase 1 <10 hours; Phase 2 can be outsourced |
| Email list open rates decline | Low | Medium | Segment by firm; send specific alerts not blasts |

## The One Metric That Matters

**Email list size** is the single leading indicator for all revenue streams.
Every traffic channel, every content piece, every tool improvement must have
exactly one email capture CTA. Not a generic "subscribe" — a specific value offer:

- Tool pages: "Get alerted when these rules change"
- Rule pages: "Get notified when this rule is updated"
- Blog posts: "Download the free rule cheat sheet"
- Course waitlist: "Join waitlist — $97 for first 50 students"

**Email list milestones that unlock revenue:**

| List Size | Unlock |
|---|---|
| 100 | Rule alert demand survey (validates Stream 3) |
| 250 | Launch rule alert paid tier ($7/month) |
| 500 | Pro tool waitlist launch email |
| 1,000 | Course pre-sell (minimum viable launch list) |
| 2,500 | Media kit ready for sponsored listings |
| 5,000 | Full newsletter monetisation; B2B outreach compelling |
